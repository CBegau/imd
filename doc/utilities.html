Utility Programs

The imd/util directory contains a number of utility programs for the preparation of IMD input files and the postprocessing of IMD output files. They are grouped into different categories:

Distribution Tools for dealing with distribution files
Configuration Tools for postprocessing IMD configuration files
Correlation Tools for postprocessing IMD correlation function files
Potential Tools for dealing with potential files
Miscellaneous Miscellaneous tools
