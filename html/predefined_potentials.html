Predefined Pair Potentials in IMD

IMD allows to use some predefined pair potentials. Currently, the following potentials are supported:

Lennard-Jones potential Morse potential Buckingham potential

By default, the potential energies and forces for these potentials are computed analytically during the simulation. In this case, cutoff radii rcij have to be specified in the parameter file through the IMD parameter r_cut. r_cut must be given for all pairs ij of atom types in the following format:

r_cut rc00 rc01 rc02 ... rc0(N-1) rc11 rc12 ... rc(N-1)(N-1)

where N is the number of atom types.

Alternatively, by specifying minimal radii through the IMD parameter r_begin in the same format as the parameter r_cut, a potential table is created at the start of the simulation and is used to compute the potential energies and forces. In that case, the numbers of tabulated points have to be specified by the integer parameter pot_res, again in the same format as the parameter r_cut. This means that the number of tabulated points can be given for each interaction between different atom types independently.

The potentials (in the analytical as well as in the tabulated form) are shifted in such a way that their values at the cutoff radii are zero.

All predefined pair potentials can be combined. However, the cutoff radii, the starting radii, and the numbers of tabulated points have to be the same for all potential forms, which is not a restriction if for each pair of atom types only one potential is used.
