Release Notes for IMD 1.2

This document describes changes of IMD 1.2 from the previous version 1.1. This description is intended for users who have already worked with IMD.

New Features

  • Mean square displacements

    IMD now allows to compute mean square displacements.

  • Correlation functions

    IMD now allows to compute van Hove (self) correlation functions. Still rather buggy.

  • New shear routines

    A new shear ensemble has been implemented.

  • Disloc option

    A new disloc option allows to selectively write certain particles only. Useful to see dislocations.

  • Improved support for sockets

    The support for sockets has been improved. These are needed for online visualization.

  • Support for PACX

    Support for PACX libraries has been added. This is needed for Metacomputing, but also useful if certain MPI routines are not available.

  • EAM potentials

    Support for EAM potentials has been added (RISC only).

  • Three body potentials

    Support for two- and three-body potentials has been added (RISC only).

Important Bug Fixes

None so far :-)
